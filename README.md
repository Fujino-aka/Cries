# Commands
/start - Start the bot

/language - Change language of the questions asked by Akinator ( This doesn't change the language of the bot interface! )

`/play` - Start playing!

`/me` - Shows stats about you.

`/leaderboard` - Check leaderboard. It includes various categories!
